# Netahougaii

睡眠催促modです。

とりあえず寝たほうがいいですよ(笑)


## 動作環境

- Minecraft 1.20.x
- Fabric API 0.88.x 以上
- modmenu 6.0.0 以上

## 使用方法<説明書>

   - 表示方法変更
   ```
    /Netahougaii set_mode <ALWAYS, SHOW_ONCE, OVERWORLD_ONLY, HIDE>
   ```
   - 音楽のボリューム
   ```
    /Netahougaii set_volume <0-100>
   ```
   - サブタイトルのテキストの変更
   ```
    /Netahougaii set_message <hour> <message>
    /Netahougaii reset_message
   ```
   
   23-2時の間一時間刻みで時報が流れます
